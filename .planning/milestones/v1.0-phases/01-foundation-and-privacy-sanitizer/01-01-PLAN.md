3. **tests/test_sanitizer.py** — Complete test file with these tests (use the patterns from 01-RESEARCH.md test section, adapted):

   **PRIV-01: test_output_is_plain_text** — HTML input with h1, p, script tags. Assert no `<` or `>` in clean_text. Assert "alert" not in output (script content stripped).

   **PRIV-02: test_tracking_pixels_removed** — HTML with `<img src="https://open.tracker.com/pixel.gif" width="1" height="1"/>`. Assert "tracker.com" not in clean_text. Assert "<img" not in clean_text.

   **PRIV-03: test_utm_params_stripped** — HTML with link containing `?utm_source=newsletter&utm_medium=email&keep=this`. Assert "utm_source" not in output. Assert "utm_medium" not in output. Assert "keep=this" in output (non-tracking param preserved).

   **PRIV-04: test_no_user_email_in_output** — HTML body containing `config.user_email` literally (e.g. "Unsubscribe: testuser@example.com"). Assert `config.user_email not in result.clean_text`. Assert `config.user_email not in result.sender_domain`.

   **PRIV-04 extra: test_no_user_name_in_output** — HTML body containing "Hi Alice Testington". Assert `config.user_name not in result.clean_text`.

   **PRIV-05: test_extra_patterns** — Config with `extra_patterns=["ACME-\\d+"]`. HTML body containing "Your code: ACME-12345". Assert "ACME-12345" not in output. Assert "[REDACTED]" in output.

   **PRIV-06: test_sender_domain_only** — Sender is `'"Morning Brew" <digest@morningbrew.com>'`. Assert `result.sender_domain == "morningbrew.com"`. Assert "@" not in result.sender_domain.

   **PRIV-07: test_truncation** — Config with max_body_chars=100. Body with 500 chars. Assert `len(result.clean_text) <= 100`.

   **PRIV-08: test_clean_message_has_no_headers** — Structural test: `assert not hasattr(CleanMessage, 'to')`, `assert not hasattr(CleanMessage, 'cc')`, `assert not hasattr(CleanMessage, 'bcc')`, `assert not hasattr(CleanMessage, 'message_id')`. Also verify CleanMessage fields are exactly: subject, sender_domain, date, clean_text (use `dataclasses.fields()`).

   **DOCS-02: test_env_example_exists** — Assert `Path(".env.example").exists()`. Read file content, assert it contains "IMAP_HOST", "IMAP_PORT", "IMAP_USERNAME", "USER_DISPLAY_NAME", "REDACT_PATTERNS", "MAX_BODY_CHARS".

   **Edge cases (add these too):**
   - **test_html_fallback_to_body_text** — body_html=None, body_text="plain content". Assert result.clean_text contains "plain content".
   - **test_empty_body** — body_html=None, body_text=None. Assert result.clean_text == "" (no crash).
   - **test_subject_pii_redacted** — Subject contains user_name "Alice Testington". Assert user_name not in result.subject (per research Open Question 3 recommendation).

   Each test creates a RawMessage, calls `sanitize(raw, config)`, and asserts on the CleanMessage result. All tests will currently fail with NotImplementedError — this is the expected RED state.
