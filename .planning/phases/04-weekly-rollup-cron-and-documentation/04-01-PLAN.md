    - format_weekly_input(digest_files: list[Path]) -> str
      - Concatenates file content with "---\nDate: YYYY-MM-DD\n\n{content}" headers
      - Date extracted from filename stem

    - weekly_archive_filename(today: date) -> str
      - Returns "weekly-{iso.year}-W{iso.week:02d}.md"
      - Uses isocalendar().year not today.year (year-boundary correctness)

    - save_weekly_archive(digest_md, config) -> Path
      - Writes to output_dir / weekly_archive_filename(date.today())
      - Creates output_dir if missing

    CLI integration (test via main()):
    - --dry-run: reports file count, does NOT call Claude, exits 0
    - No daily files found: exits 2
    - Claude CLI error (RuntimeError): exits 3
    - Claude CLI missing (FileNotFoundError): exits 3
    - Config error (ValueError): exits 1
    - Success with output_format=email: calls send_digest_email
    - Success with output_format=markdown: saves weekly archive only
